# Лендинг автоматизации маркетинга и продаж

Современный адаптивный лендинг, созданный с помощью React + Vite.

## Технологии

- React 18
- Vite 5
- CSS Modules
- Современная типографика (Inter)
- Адаптивный дизайн

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

## Сборка для продакшена

```bash
npm run build
```

## Деплой на GitHub Pages

1. Убедитесь, что в `vite.config.js` указан правильный `base` путь (имя репозитория)
2. Запустите сборку: `npm run build`
3. Файлы из папки `dist` можно задеплоить на GitHub Pages

### Автоматический деплой через GitHub Actions

В репозитории уже настроен workflow для автоматического деплоя на GitHub Pages при пуше в ветку `main`.

## Структура проекта

```
/src
  /components      - React компоненты
  /styles          - Глобальные стили
  /assets          - Изображения и шрифты
```

## Адаптивность

Сайт полностью адаптивен и корректно отображается на:
- 1920px+
- 1440px
- 1024px
- 768px
- 480px
- 360px

## Особенности

- Плавный скролл по якорям
- Анимации появления секций при скролле
- Hover-эффекты на интерактивных элементах
- Минималистичный современный дизайн
- Оптимизированная производительность
